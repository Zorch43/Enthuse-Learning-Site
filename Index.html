Our Product

We are currently developing a suite of language learning software applications that will allow language learners to:

  • Learn new vocabulary words
  • Learn phrases used in everyday situations
  • Learn grammar and sentence structure

Language lessons will be available in multiple languages. Language learners will also be able to add content (words, phrases, grammatical structures) and create their own lessons. We plan to release the following versions of the software:

  • PC
  • Web
  • Mobile

The software is open source. If you would like to contribute to the project, go to the Source Forge site.
